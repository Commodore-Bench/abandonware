
; BUG

STING

  AIM TO RADAR
  IF RADAR>0 GOTO SPIN
  0-RADAR TO SHOT
  IF DAMAGE<100 GOTO START
  GOTO STING

SPIN
  AIM+191 TO AIM
  IF DAMAGE<100 GOTO START
  GOTO STING

START
  IF Y>120 GOTO START2
  246-Y*3 TO SPEEDY
  GOTO DOWN

START2
  10-Y*3 TO SPEEDY
  GOTO UP

RIGHT
  0 TO SPEEDY
  246-X*2 TO SPEEDX
  85 TO A
  242-X TO RANDOM
  RANDOM TO Z

CRAWL
  GOSUB EYES
  IF Z-X<14 GOTO DOWN
  250-X*2 TO SPEEDX
  GOTO CRAWL

DOWN
  0 TO SPEEDX
  246-Y*2 TO SPEEDY
  175 TO A
  242-Y TO RANDOM
  RANDOM TO Z

SCUTTLE
  GOSUB EYES
  IF Z-Y<14 GOTO LEFT
  250-Y*2 TO SPEEDY
  GOTO SCUTTLE

LEFT
  0 TO SPEEDY
  10-X*2 TO SPEEDX
  X TO RANDOM
  RANDOM+14 TO Z
  265 TO A

CREEP
  GOSUB EYES
  IF X-Z<14 GOTO UP
  6-X*2 TO SPEEDX
  GOTO CREEP

UP
  0 TO SPEEDX
  10-Y*2 TO SPEEDY
  355 TO A
  Y TO RANDOM
  RANDOM+14 TO Z

SCAMPER
  GOSUB EYES
  IF Y-Z<14 GOTO RIGHT
  6-Y*2 TO SPEEDY
  GOTO SCAMPER

EYES
  B+1 TO B
  A TO AIM TO RADAR
  IF RADAR<0 GOSUB STOPLEGS
  AIM+180 TO AIM TO RADAR
  IF RADAR<0 GOSUB STOPLEGS
  AIM+190 TO AIM TO RADAR
  IF B=7 GOTO JUMP
  IF RADAR<0 GOSUB STOPLEGS
  AIM+180 TO AIM TO RADAR
  IF RADAR<0 GOSUB STOPLEGS
  ENDSUB

STOPLEGS
  0 TO SPEEDX TO SPEEDY
  20-RADAR TO SHOT

BITE
  AIM TO RADAR
  IF RADAR>0 GOTO GOLEGS
  0-RADAR TO SHOT
  GOTO BITE

GOLEGS
  IF A=85 246-X*2 TO SPEEDX
  IF A=355 10-Y*2 TO SPEEDY
  IF A=265 10-X*2 TO SPEEDX
  IF A=175 246-Y*2 TO SPEEDY
  ENDSUB

JUMP
  0 TO B
  ENDSUB
