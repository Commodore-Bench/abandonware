
              Usage Notes


To save or restore the game during
gameplay, type SR.

To see historical notes about your
current location, type H.

The password to play part 2 is
MALINCHE.

Despite being available from the boot
menu, saving the game before starting
the game will create a condition where
it does not recognize the program disk,
forcing you to reboot. This bug was
also present in the original game.
